# √ Ejercicio 14 — ft_sqrt

Objetivo: devolver la raíz cuadrada entera si existe, o `0` si no es un cuadrado perfecto.

---

- 📁 Directorio de entrega: `ex14/`
- 📄 Archivos a entregar: `ft_sqrt.c`
- ✅ Funciones autorizadas: Ninguna

---

### Enunciado

Crea una función que devuelva la raíz cuadrada de un número (si existe), o `0` si la raíz cuadrada es un número irracional.

Prototipo:

```c
int ft_sqrt(int nb);
```

Nota: la función debe devolver su resultado en menos de dos segundos.
