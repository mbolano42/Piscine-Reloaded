# 🔁 Ejercicio 25 — ft_foreach

Objetivo: aplicar una función a cada elemento de un array de enteros.

---

- 📁 Directorio de entrega: `ex25/`
- 📄 Archivos a entregar: `ft_foreach.c`
- ✅ Funciones autorizadas: Ninguna

---

### Enunciado

Crea la función `ft_foreach` que, para un array de enteros dado, aplique una función `f` a todos sus elementos siguiendo el orden del array.

Prototipo:

```c
void ft_foreach(int *tab, int length, void (*f)(int));
```

Ejemplo de uso: `ft_foreach(tab, 1337, &ft_putnbr);`
