# 🔎 Ejercicio 03 — find_sh.sh

✨ Objetivo: listar todos los archivos terminados en `.sh` mostrando solo el nombre sin la extensión.

---

- 📁 Directorio de entrega: `ex03/`
- 📄 Archivos a entregar: `find_sh.sh`
- ✅ Funciones autorizadas: Ninguna

---

### 📝 Enunciado

Escribe una línea de comando que busque todos los nombres de archivo acabados en `.sh` en el directorio actual y todos sus subdirectorios. Debe mostrar solo los nombres de archivo sin `.sh`.

Ejemplo de salida:

```bash
$>./find_sh.sh | cat -e
find_sh$
file1$
file2$
file3$
$>
```
