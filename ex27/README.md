# 📄 Ejercicio 27 — display_file

Objetivo: crear un programa `ft_display_file` que muestre el contenido de un archivo pasado como argumento.

---

- 📁 Directorio de entrega: `ex27/`
- 📄 Archivos a entregar: `Makefile` y todos los archivos necesarios para el programa
- ✅ Funciones autorizadas: `close`, `open`, `read`, `write`

---

### Enunciado

Crea un programa llamado `ft_display_file` que muestre, en la salida estándar, solo el contenido del archivo dado como argumento.

- El directorio de entrega debe tener un `Makefile` con las reglas `all`, `clean`, `fclean`.
- El binario se llamará `ft_display_file`.
- La función `malloc` está prohibida; declara un array de tamaño fijo para leer.
- Todos los archivos dados como argumento serán válidos.
- Mensajes de error:
  - Si no se da ningún argumento: `File name missing.`
  - Si hay más de un argumento: `Too many arguments.`
  - Si el archivo no puede leerse: `Cannot read file.`
