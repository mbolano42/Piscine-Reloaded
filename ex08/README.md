# Ejercicio 08 — ft_is_negative

Objetivo: mostrar `N` o `P` dependiendo del signo de un entero.

---

- Directorio de entrega: `ex08/`
- Archivos a entregar: `ft_is_negative.c`
- Funciones autorizadas: `ft_putchar`

---

### Enunciado

Crea una función que muestre `N` o `P` dependiendo del signo del entero introducido como parámetro.

- Si `n` es negativo, muestra `N`.
- Si `n` es positivo o cero, muestra `P`.

Prototipo:

```c
void ft_is_negative(int n);
```
