# ⚖️ Ejercicio 22 — ft_abs.h

Objetivo: crear una macro `ABS` que devuelva el valor absoluto de su argumento.

---

- 📁 Directorio de entrega: `ex22/`
- 📄 Archivos a entregar: `ft_abs.h`
- ✅ Funciones autorizadas: Ninguna (se permite macro)

---

### Enunciado

Crea una macro `ABS` que sustituya su argumento por su valor absoluto. Ejemplo seguro:

```c
#define ABS(Value) ((Value) < 0 ? -(Value) : (Value))
```

(Se permite esta excepción a la Norma para practicar macros.)
