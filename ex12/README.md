# 🔁 Ejercicio 12 — ft_iterative_factorial

Objetivo: implementar el factorial de forma iterativa.

---

- 📁 Directorio de entrega: `ex12/`
- 📄 Archivos a entregar: `ft_iterative_factorial.c`
- ✅ Funciones autorizadas: Ninguna

---

### Enunciado

Crea una función iterativa que devuelva el factorial del número introducido como parámetro. Si hay un error, la función debe devolver `0`.

Prototipo:

```c
int ft_iterative_factorial(int nb);
```

Nota: la función debe devolver su resultado en menos de dos segundos.
