# 🏊‍♂️ Piscine Reloaded — Índice y guía

¡Bienvenido/a a la colección de ejercicios de la **Piscine Reloaded**! 🎉

Este repositorio contiene los ejercicios `ex00` → `ex27`. Cada ejercicio tiene su propio `README.md` con el enunciado, requisitos y enlaces de navegación (anterior / siguiente).

---

## ✨ Introducción

Este conjunto de ejercicios te ayudará a repasar los fundamentos de C y algunos comandos de Unix: funciones, punteros, estructuras, Makefiles y utilidades de shell. Lee cada `README.md` dentro de la carpeta del ejercicio antes de empezar.
