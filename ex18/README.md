# 📎 Ejercicio 18 — ft_print_params

Objetivo: programa que muestre sus argumentos, uno por línea.

---

- 📁 Directorio de entrega: `ex18/`
- 📄 Archivos a entregar: `ft_print_params.c`
- ✅ Funciones autorizadas: `ft_putchar`

---

### Enunciado

Esto es un programa, así que deberías tener una función `main` en tu archivo `.c`.

Crea un programa que muestre sus argumentos dados, cada uno en una línea.

Ejemplo:

```bash
$>./a.out test1 test2 test3
test1
test2
test3
$>
```
