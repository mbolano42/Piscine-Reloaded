# 🔁 Ejercicio 13 — ft_recursive_factorial

Objetivo: implementar el factorial de forma recursiva.

---

- 📁 Directorio de entrega: `ex13/`
- 📄 Archivos a entregar: `ft_recursive_factorial.c`
- ✅ Funciones autorizadas: Ninguna

### Enunciado

Crea una función recursiva que devuelva el factorial del número introducido como parámetro. Si hay un error, la función debe devolver `0`.

Prototipo:

```c
int ft_recursive_factorial(int nb);
```
