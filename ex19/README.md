# 🔣 Ejercicio 19 — ft_sort_params

Objetivo: ordenar los argumentos pasados al programa y mostrarlos en orden ASCII.

---

- 📁 Directorio de entrega: `ex19/`
- 📄 Archivos a entregar: `ft_sort_params.c`
- ✅ Funciones autorizadas: `ft_putchar`

---

### Enunciado

Esto es un programa, así que deberías tener una función `main`.

Crea un programa que muestre sus argumentos dados ordenados en orden ASCII. El programa debe mostrar todos los argumentos, excepto `argv[0]`, y cada argumento debe terminar con un salto de línea.
