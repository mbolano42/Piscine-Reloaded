# 📚 Ejercicio 21 — ft_range

Objetivo: crear una función que devuelva un array de enteros entre `min` (inclusive) y `max` (exclusive).

---

- 📁 Directorio de entrega: `ex21/`
- 📄 Archivos a entregar: `ft_range.c`
- ✅ Funciones autorizadas: `malloc`

---

### Enunciado

Crea una función `ft_range` que devuelva un array de enteros que contenga todos los valores entre `min` y `max` (min incluido, max excluido).

Prototipo:

```c
int *ft_range(int min, int max);
```

Si `min >= max`, debe devolverse un puntero nulo.
