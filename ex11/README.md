# ➗ Ejercicio 11 — ft_div_mod

Objetivo: implementar `ft_div_mod()` que calcule división y resto.

---

- 📁 Directorio de entrega: `ex11/`
- 📄 Archivos a entregar: `ft_div_mod.c`
- ✅ Funciones autorizadas: Ninguna

---

### Enunciado

Crea una función `ft_div_mod` con el siguiente prototipo:

```c
void ft_div_mod(int a, int b, int *div, int *mod);
```

Esta función divide `a` entre `b` y almacena el resultado en `*div`. Además, almacena el resto en `*mod`.
