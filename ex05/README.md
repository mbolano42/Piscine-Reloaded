# 🧪 Ejercicio 05 — ¿Puedes crearlo?

✨ Objetivo: crear un archivo con un nombre especial que contenga únicamente el texto `42`.

---

- 📁 Directorio de entrega: `ex05/`
- 📄 Archivos a entregar: `"\?$*'MaRViN'*$?\"` (nombre exacto)
- ✅ Funciones autorizadas: Ninguna

---

### 📝 Enunciado

Crea un archivo que contenga únicamente `42`, y NADA más. Su nombre será exactamente:

```
"\\?$*'MaRViN'*$?\\"
```

Ejemplo de comprobación:

```bash
$>ls -lRa *MaRV* | cat -e
-rw---xr-- 1 75355 32015 2 Oct 2 12:21 "\?$*'MaRViN'*$?\"$
$>
```
